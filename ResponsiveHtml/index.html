Description

Using valid HTML and CSS, create a document with a three column layout. The left column should have a width of 170px and the right column should be 200px wide. The center column should fluidly expand to fill the center, but should always be at least 350px wide. When the window or device is not large enough to hold all 3 columns, the columns should each fill the entire width of the screen, left column first, center column next and right column last. The layout should work correctly in the major browsers (Chrome, Firefox, Safari, and IE9) and in the major mobile and tablet (Android and iOS) devices.
